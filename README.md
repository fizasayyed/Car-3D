# Car-3D

A 3D car using three.js.

## Run Locally

Clone the project

```bash
  git clone https://github.com/fizasayyed/Metaverse-Final.git
```

Go to the project directory

Install lite-server or extension Live Server in VS Code

```bash
 npm install --global lite-server
```

Start the server

```bash
  lite-server.cmd
```
